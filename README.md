# template-vite-vue-ts-tailwind-jit

- **template of:** vite + vue + typescript
- **tailwindcss:** `^2.2.19`

## Started
with npm:
```bash
npm install
```
with yarn:
```bash
yarn
```

---
## Develop
with npm:
```bash
npm run dev
```
with yarn:
```bash
yarn dev
```

---
## Build
with npm:
```bash
npm run build
```
with yarn:
```bash
yarn build
```
